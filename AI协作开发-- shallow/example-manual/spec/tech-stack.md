### 后端目录结构

```
backend/
├── app/               # 应用主目录
│   ├── api/           # API 路由
│   ├── models/        # 数据模型
│   ├── schemas/       # Pydantic 模型
│   └── main.py        # 应用入口
├── core/              # 核心业务逻辑
├── services/          # 服务层

├── db/                # 数据库
└── config/            # 配置文件
```

**约束**：
- API 路由必须放在 `app/api/`，不得放在其他目录
- 业务逻辑必须放在 `services/` 或 `core/`，不得在路由中直接写逻辑
- 工具函数必须放在 `utils/`，不得重复造轮子
