# AgentHub 面试准备材料

本目录包含围绕 AgentHub 项目（后端为主）的面试准备材料，按模块组织。

## 文件列表

| 序号 | 文件 | 模块 | 内容概要 |
|------|------|------|---------|
| 01 | `01-项目总览与架构.md` | 项目总览 | 整体架构、技术选型、设计模式、容错、部署 |
| 02 | `02-Orchestrator编排器.md` | 编排器 | 路由决策、复杂度分类、流式输出、Fallback、Skill注入 |
| 03 | `03-记忆模块.md` | 记忆系统 | 5层记忆框架、提取/检索/衰减、用户画像、特性开关 |
| 04 | `04-RAG模块.md` | RAG | 文档入库/去重、ChromaDB、嵌入模型、检索质量 |
| 05 | `05-Agent系统.md` | Agent | BaseAgent/CustomAgent、A-Tier配置、ReAct、Agent Builder |
| 06 | `06-LLM后端层.md` | LLM后端 | 多后端抽象、健康检查、流式/非流式、错误处理 |
| 07 | `07-LangGraph工作流与动态规划.md` | 工作流 | StateGraph、条件边、Checkpointer、动态规划、重规划 |
| 08 | `08-数据库与数据模型.md` | 数据库 | ORM模型、SQLite选择、Schema迁移、JSON字段、事务 |
| 09 | `09-API与通信层.md` | API通信 | REST/WebSocket/SSE、JWT鉴权、流式队列、安全 |
| 10 | `10-配置-Skill系统与扩展性.md` | 配置扩展 | 配置管理、双轨Skill、Prompt管理、扩展点、微服务拆分 |

## 使用方式

每个文件的结构：
- **Q0**：请简单介绍一下你的项目/xx模块 → 用于开场自我介绍
- **Q1-Q20**：面试官可能追问的 20 个技术问题 → 每个问题都附有参考答复

面试前建议：
1. 先通读所有文件，建立对项目全局的清晰认知
2. 重点准备 Q0（每个模块的介绍），这是面试的"开场白"
3. 针对自己最熟悉的模块深入准备
4. 回答时结合代码细节（行号、函数名、具体实现），展现真实参与度
